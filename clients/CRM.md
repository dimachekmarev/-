# CRM — Статусы и воронка

## Этапы работы с клиентом

| Статус | Описание |
|--------|----------|
| 🔴 Лид | Первый контакт, нужен intake |
| 🟠 Intake | Заполнен опросник, ждёт анализа |
| 🟡 Анализ | Декомпозиция процессов, скоринг |
| 🟢 Пилот | Настройка первого агента, 2-4 недели |
| 🔵 Контракт | Полноценное внедрение |
| ⚪ Поддержка | Сопровождение |

## Правила

- Каждый клиент = MD-файл в [[Клиенты]]
- Каждый проект = MD-файл в [[Проекты]]
- Статус обновляется при переходе между этапами
- Все контакты с клиентом фиксируются в его карточке
