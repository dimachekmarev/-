# 2026-05-19 — Hermes Agent: голосовое управление через Telegram

**Источник:** Настройка системы (Hermes Agent)

## Инсайт

Hermes Agent поддерживает распознавание голосовых сообщений (STT) через `faster-whisper` с полной поддержкой русского языка. Модель `small` даёт хорошее качество при минимальных затратах ресурсов.

## Почему это важно

Можно диктовать мысли голосом в Telegram — агент их распознаёт и выполняет задачи. Это быстрее, чем печатать, особенно для длинных описаний.

## Применение

- Диктовка задач и идей
- Голосовые заметки вместо текстовых
- Быстрая передача контекста агенту
