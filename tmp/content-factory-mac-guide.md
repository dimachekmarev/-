# Контент-завод на Mac — что поставить и как запустить

Создано: 2026-06-09
Для: друг Дмитрия
Смысл: превратить Mac в машину, которая сама делает контент

---

## Базовая философия

Контент-завод = цепочка из трёх станций:

```
СЫРЬЁ  →  ПРОИЗВОДСТВО  →  ВЫХОД
(идеи)    (AI + монтаж)    (публикация)
```

Каждая станция автоматизируется отдельно. Не надо автоматизировать всё сразу.

---

## СТАНЦИЯ 1 — Сырьё (идеи и тренды)

### Что поставить (бесплатно):

| Инструмент | Зачем | Ссылка |
|------------|-------|--------|
| **Hermes Agent** | AI-агент для всего: поиск трендов, генерация текстов, кроны | `curl -fsSL https://hermes-agent.nousresearch.com/install.sh \| bash` |
| **Obsidian** | База знаний, хранение идей, связывание заметок | [obsidian.md](https://obsidian.md) |
| **Mac Shortcuts** | Автоматизация сбора ссылок, скриншотов, заметок | встроено в Mac |

### Автоматизация:

1. Hermes-крон собирает тренды каждое утро → пишет в Obsidian
2. Mac Shortcuts сохраняет ссылки из Telegram/YouTube в одну папку
3. Раз в день — 15-минутный обзор собранного → выбор темы

```bash
# Пример: крон для сбора трендов
hermes cron create \
  --name "Утренний сбор трендов" \
  --schedule "0 9 * * *" \
  --prompt "Собери 10 трендов по теме [ТВОЯ НИША]. Запиши в файл."
```

## СТАНЦИЯ 2 — Производство (AI создаёт контент)

### Для видео (Shorts / Reels / Long):

| Инструмент | Зачем | Цена |
|------------|-------|------|
| **CapCut Desktop** | Быстрый монтаж Shorts: субтитры, авто-обрезка, эффекты | бесплатно |
| **DaVinci Resolve** | Серьёзный монтаж Long-видео, цветокоррекция | бесплатно |
| **OBS Studio** | Запись экрана, стриминг | бесплатно |
| **HandBrake** | Сжатие видео под площадки | бесплатно |
| **ElevenLabs** | Озвучка AI-голосом (без лица) | бесплатный тариф |

### Для текстов и постов:

| Инструмент | Зачем |
|------------|-------|
| **Hermes Agent** | Пишет сценарии, посты, описания, заголовки |
| **Claude Desktop** | Глубокие тексты, аналитика |
| **ChatGPT Desktop** | Быстрые черновики |

### Для визуала (без лица):

| Инструмент | Зачем |
|------------|-------|
| **Canva** | Обложки, превью, дизайн постов |
| **Midjourney** (через Discord) | Генерация изображений под контент |
| **ComfyUI** (локально) | Бесплатная генерация изображений, если Mac на M1/M2 |

### Автоматизация цепочки:

Через **Hermes Agent** — один скрипт, который:
1. Берёт тему из Obsidian
2. Генерит сценарий (AI)
3. Создаёт визуал (Canva API / Midjourney)
4. Собирает всё в папку «Готово к публикации»

```bash
# Автоматический выпуск Shorts-сценария
hermes cron create \
  --name "Ежедневный сценарий Shorts" \
  --schedule "0 10 * * *" \
  --prompt "Создай сценарий Shorts на 60 секунд по теме [ТВОЯ НИША]. 
  Формат: хук → проблема → решение → CTA. 
  Добавь 3 варианта заголовка. Сохрани в Obsidian."
```

## СТАНЦИЯ 3 — Выход (публикация и аналитика)

### Для публикации:

| Инструмент | Куда постит | Цена |
|------------|-------------|------|
| **Buffer** | YouTube, TikTok, Instagram, X | бесплатный тариф |
| **Metricool** | Всё + аналитика | бесплатный тариф |
| **Hermes Agent** | Telegram, VK, Discord | бесплатно |

### Автоматизация:

```bash
# Автопостинг в Telegram-канал
hermes cron create \
  --name "Автопост в Telegram" \
  --schedule "0 12,18 * * *" \
  --prompt "Возьми последний готовый пост из Obsidian и отправь в Telegram."
```

## МИНИМАЛЬНЫЙ СТАРТОВЫЙ НАБОР (бесплатный)

Если ставить только необходимое прямо сейчас:

1. **Hermes Agent** — мозг всего
2. **CapCut Desktop** — монтаж Shorts
3. **Canva** — обложки и визуал
4. **Obsidian** — база идей и сценариев
5. **Telegram** — канал для выхода

Этого хватит, чтобы выпускать **2 Shorts в день + 3 поста в Telegram** полностью на автопилоте.

## ПРИМЕР ЖИВОЙ ЦЕПОЧКИ (как у Дмитрия)

```
08:00 — Hermes собирает тренды дня → Obsidian
09:00 — Hermes генерит 2 сценария Shorts → Obsidian
10:00 — Hermes генерит 2 поста для Telegram/VK → Obsidian
12:00 — Автопост в Telegram
14:00 — Автопост в VK
18:00 — Вечерний пост в Telegram
20:00 — Отчёт дня: просмотры, охваты, что сработало
```

**Результат:** 4-6 публикаций в день, 0 ручной работы.

## КАК НАЧАТЬ ПРЯМО СЕГОДНЯ

1. Поставить Hermes Agent (5 минут)
2. Создать папку в Obsidian «Контент-завод» (2 минуты)
3. Запустить первый крон — сбор трендов на завтра (3 минуты)
4. Поставить CapCut (пока скачивается — написать первый сценарий)

Итого: **30 минут до первого автоматического контента.**
