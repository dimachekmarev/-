## Что такое cron-задачи

Это **автоматические процессы**, которые запускаются по расписанию без участия человека. Как будильник: поставил на 07:00 — и каждое утро что-то происходит.

У нас их **46 активных** + 40 приостановленных.

Каждая задача бывает одного из двух типов:
- **🤖 LLM-задача** — ИИ-агент получает промпт и выполняет его (думает, ищет, пишет)
- **📜 Скрипт** — просто запускается готовый код без участия ИИ

Результат доставляется:
- `origin` — Дмитрию в Telegram
- `local` — сохраняется локально на сервере, без уведомлений

## 🔧 Как это работает технически

### Платформа
Всё крутится на **Hermes Agent** — это фреймворк для AI-агентов.  
Сервер: VPS, 1GB RAM, 2 ядра.  
Задачи управляются встроенным cron-планировщиком.

### Типы доставки
- **origin** → приходит Дмитрию в Telegram
- **local** → сохраняется на сервере (не спамит)
- Большинство задач — `local`, не беспокоят

### Как добавить свою задачу
```bash
hermes cron create \
  --name "Моя задача" \
  --schedule "0 9 * * *" \
  --prompt "Что нужно сделать агенту"
```

### Как посмотреть все задачи
```bash
hermes cron list --all
```

### Как остановить/запустить
```bash
hermes cron pause <id>
hermes cron resume <id>
```

## 📊 Общая статистика

- **46 активных задач**
- **40 приостановленных** (были отключены для снижения шума)
- **~30 LLM-задач** используют AI
- **~16 скриптовых** работают без AI

**Самый плотный час:** 07:00–10:00 утра (запускается ~15 задач)
**Самый тихий:** 22:00–03:00 ночи

## 🎯 Что умеет система в двух словах

- Сама пишет и публикует контент (VK, сайт, блоги)
- Сама ищет тренды и денежные идеи
- Сама следит за здоровьем сервера
- Сама синхронизирует память и Obsidian
- Сама чинит себя при сбоях
- Сама готовит утренний отчёт Дмитрию

**Дмитрию остаётся только принимать решения и делать продажи.**
