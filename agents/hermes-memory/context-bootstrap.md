## Где брать контекст

1. Память/профиль: `agents/hermes-memory/USER.full.md` и `agents/hermes-memory/MEMORY.full.md`.
2. Контракт памяти: `agents/Memory-Contract-Hermes-Default.md`.
3. Операционный дневник: `meetings/Единый дневник операций.md`.
4. Проекты: `projects/`.
5. Агенты и структура: `agents/Агенты.md`.
6. Автоматизации: `automation/`.
7. Sync status: `agents/hermes-memory/sync-status.json`.
