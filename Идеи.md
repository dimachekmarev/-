# Идеи Дмитрия

Это входящий файл для быстрых идей.

## Правило
Когда Дмитрий пишет **«ИДЕЯ»** или **«ЕСТЬ ИДЕЯ»**, идея просто сохраняется сюда.

Важно:
- не выполнять сразу;
- не запускать агентов;
- не делать план реализации без отдельной команды;
- по запросу показать список идей, чтобы Дмитрий выбрал, с какой начинать.

## Формат записи
```markdown
### YYYY-MM-DD — короткое название
- Статус: входящая
- Текст идеи: ...
- Источник: Telegram
- Следующее действие: ждать выбора Дмитрия
```
