# Pinterest system для Дмитрия Чекмарева

Цель: собрать Pinterest как тихий источник трафика для обучалок, AI-аватаров, сайта, медодежды и онлайн-империи.

## Что сделано

1. Секреты Pinterest сохранены локально в env-файл с правами 600.
2. Собрана структура досок.
3. Собран CSV первых пинов.
4. Подготовлена схема SEO и воронок.

## Секреты

Файл:
/root/.config/dima-chekmarev/pinterest.env

Не публиковать в чат, GitHub, Obsidian или сайт.

## Первый запуск

1. Создать доски Pinterest из docs/boards.md.
2. Заменить link в CSV на реальный сайт/лендинг.
3. Добавить реальные изображения пинов.
4. Запустить dry-run через social-media-pinterest pipeline.
5. После проверки — публиковать с задержкой 60+ секунд.

## Воронка

Pin → бесплатная обучалка → Telegram/сайт → лид-магнит → платный пакет → кейс → новый pin.

# Что нужно для реализации Pinterest

1. Реальный URL сайта или лендинга вместо временного example.com.
2. Pinterest аккаунт с созданными досками.
3. Изображения для пинов 1000x1500 или 1080x1920.
4. Решить язык: русский для РФ/СНГ или английский для большего RPM.
5. Не постить слишком быстро: задержка 60+ секунд.
6. Первые 20-50 пинов делать мягкими обучающими, не агрессивно продающими.
7. Проверить, разрешает ли текущий Pinterest app нужные scopes для API. Если нет — используем Selenium-пайплайн.
