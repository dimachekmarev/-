# Pinterest system для Дмитрия Чекмарева

Цель: Pinterest как тихий источник трафика для обучалок, AI-аватаров, сайта, медодежды и онлайн-империи.

Секреты сохранены локально:
/root/.config/dima-chekmarev/pinterest.env

Не публиковать env в чат, GitHub, Obsidian или сайт.

Воронка:
Pin -> бесплатная обучалка -> Telegram/сайт -> лид-магнит -> платный пакет -> кейс -> новый pin.

# Что нужно для реализации Pinterest

1. Реальный URL сайта или лендинга вместо временного example.com.
2. Pinterest аккаунт с созданными досками.
3. Изображения для пинов 1000x1500 или 1080x1920.
4. Выбрать язык: русский для РФ/СНГ или английский для большего RPM.
5. Не постить быстро: задержка 60+ секунд.
6. Первые 20-50 пинов делать обучающими, не агрессивно продающими.
7. Проверить scopes Pinterest app. Если API не даст публикацию — используем Selenium-пайплайн.
8. Так как секрет был отправлен в чат, лучше потом сбросить секрет приложения и заменить env.
