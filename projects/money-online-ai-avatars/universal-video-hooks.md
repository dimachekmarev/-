УНИВЕРСАЛЬНОЕ ВИДЕО ДЛЯ МНОГИХ ХУКОВ

Цель:
один рекламный видео-каркас, который можно быстро адаптировать под разные первые фразы, ниши и офферы.

Формат:
- вертикальное 9:16
- 25-35 секунд
- подходит для Shorts, Reels, TikTok, VK Клипы, Pinterest Idea Pins
- основной вариант: Higgsfield
- запасной вариант: HyperFrames локально

Структура ролика:
1. 0-2 сек — сильный хук
2. 2-7 сек — боль аудитории
3. 7-15 сек — решение через систему Дмитрия
4. 15-24 сек — демонстрация результата
5. 24-32 сек — CTA

Визуальная идея:
человек стоит в центре цифровой операционной комнаты. Вокруг него всплывают карточки: контент, деньги, задачи, сайт, аватары, Pinterest, заявки. Хаос превращается в управляемую систему.

Почему это универсально:
меняется только хук, ниша и CTA. Визуальная логика остается одинаковой: хаос → система → результат → действие.

Файлы:
- hooks.json — банк хуков
- script-template.md — универсальный сценарий
- higgsfield-master-prompt.md — главный prompt для Higgsfield
- higgsfield-prompts/ — готовые prompts под каждый хук
- hyperframes-template/ — локальный fallback-шаблон


УНИВЕРСАЛЬНЫЙ СЦЕНАРИЙ

[0-2 сек]
{HOOK}

[2-7 сек]
Большинство людей пытаются расти через хаос: посты отдельно, сайт отдельно, заявки отдельно, задачи отдельно.

[7-15 сек]
Я собираю это в одну систему: контент, AI-аватары, сайт, Pinterest, воронки и автоматизация работают вместе.

[15-24 сек]
В итоге появляется не просто красивая картинка, а поток: внимание → доверие → заявка → продажа.

[24-32 сек]
{CTA}

Тон:
спокойно, уверенно, дорого, без крика.

Музыка:
