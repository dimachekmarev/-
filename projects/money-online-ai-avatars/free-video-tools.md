Цель: делать контент максимально на халяве, не жечь токены и деньги.

## Приоритет инструментов

1. Higgsfield
- использовать для рекламных роликов, если есть бесплатные лимиты
- сильная сторона: cinematic, motion, рекламная подача

2. Seedance 2.0
- по свежим источникам: много бесплатных кредитов / иногда без аккаунта
- использовать для массовых вариаций хуков

3. Hailuo AI
- ежедневные бесплатные кредиты
- использовать для коротких 5-10 секундных сцен

4. Vidu AI
- бесплатные генерации + бонусы
- использовать для тестов персонажей и коротких переходов

5. Luma Dream Machine
- ограниченный бесплатный лимит
- использовать для самых красивых hero-сцен

6. Kling
- использовать для качества и движения персонажа, если лимит доступен

7. Open-Higgsfield-AI
- проверить как open-source/hosted вариант
- если реально работает бесплатно, сделать основным для экспериментов

8. HyperFrames локально
- fallback без внешних лимитов
- текст, графика, premium motion, карточки, обложки

## Правило экономии

Не тратить модель на то, что можно сделать шаблоном.

Порядок производства:
1. один универсальный сценарный каркас
2. 10-30 хуков
3. одна визуальная сцена
4. разные CTA
5. выгрузка пачкой

## Что делаем для Дмитрия

Контент-сериал:
“С нуля строим онлайн-империю”.

Формат:
- короткие ролики
- дневник строительства
- не понты, а путь
- каждый выпуск показывает маленький реальный шаг

Основная драматургия:
хаос → система → первые активы → контент → заявки → продукты → команда → империя.
