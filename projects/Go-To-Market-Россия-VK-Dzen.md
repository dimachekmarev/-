# Go-To-Market: Россия (VK + Dzen + Telegram + GitHub)

## Главная цель (90 дней)
1. Быстро нарастить узнаваемость в RU-сегменте.
2. Превращать охваты в деньги: лиды -> диагностики -> внедрения Smart Agent AI.
3. Построить автопилот-контур с минимальным участием Дмитрия (только approval важных решений).

## Денежная модель (в порядке приоритета)
1. Основной доход: внедрение Smart Agent AI (услуги/проекты).
2. Средний слой: консалтинг/аудит автоматизации (диагностика + roadmap).
3. Верхний слой: интеграции/партнерки/спонсорские размещения.
4. Дополнительно: обучение, шаблоны, платные разборы.

## Канальный приоритет (RU first)
1. Telegram — прогрев, доверие, диалоги.
2. VK — массовый RU-охват и регулярный контент-ритм.
3. Dzen — SEO/длинный хвост, длинные статьи.
4. GitHub — доказательство экспертности (кейсы, архитектуры, шаблоны).

## Единая воронка (перелинковка)
Каждая площадка должна вести в другие:
- VK пост -> CTA в Telegram + ссылка на кейс в GitHub
- Dzen статья -> CTA в Telegram + «тех. приложение» в GitHub
- Telegram пост -> ссылка на VK/Dzen расширенную версию
- GitHub update -> CTA в Telegram для разбора под бизнес

## Архитектура отдела (агенты)
### L0 Управление
- Chief Brand Agent: стратегия недели, фокус, приоритет каналов.
- QA/Risk Agent: проверка фактов, обещаний, репутационных рисков.

### L1 Производство
- Idea Scout RU: поиск тем под RU-рынок (боль SMB, кейсы, новости).
- VK Writer: короткие и средние форматы под VK.
- Dzen Writer: лонгриды и объясняющие статьи.
- Telegram Writer: прогрев, авторская позиция, лидогенерация.
- Repurposer: 1 идея -> 4 формата.

### L2 Продажи и аналитика
- Offer Agent: оффер недели, CTA, «входной продукт».
- Funnel Analyst: метрики по каналам, что масштабировать.
- Community Draft Agent: черновики ответов/диалогов.

## Approval Gates (подтверждает Дмитрий)
1. Оффер недели и ценовая подача.
2. Кейсы с цифрами/обещаниями/клиентскими данными.
3. Любые спорные репутационные или юридические формулировки.

## KPI (еженедельно)
- Контент: 5-7 единиц/нед по RU-каналам.
- Лиды: входящие диалоги в Telegram.
- Конверсия: диалог -> диагностика -> коммерческое предложение.
- Доход: выручка от внедрений Smart Agent AI.

## Поэтапный запуск
### Этап 1 (7 дней)
- Запуск контент-контура VK + Dzen (черновики + календарь + QA).
- Перелинковка шаблонов между каналами.
- Единый weekly оффер.

### Этап 2 (14-30 дней)
- Автоматизация публикации (где API позволяет) или semi-auto через approval.
- Ежедневный отчёт по контенту, weekly отчёт по деньгам.

### Этап 3 (30-90 дней)
- Усиление лучших рубрик.
- Партнерские интеграции и рекламные размещения.
- Подготовка к масштабированию на мировой рынок.

## Принципы качества
- Без хайпа и «магических обещаний».
- Всегда: проблема -> решение -> ограничение -> практический вывод.
- Контент строит доверие и ведёт к продаже, а не к пустому охвату.
