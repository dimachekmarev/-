# Проект 04 — Telegram-система

Цель: сделать Telegram центральным хабом лидов и продаж.

## Фокус
- Входящий поток из YouTube/TikTok
- Авто-ответы
- Скрипты дожима
- Контроль диалогов

## KPI
- Новые диалоги
- Ответы на кодовые слова
- Доля доведения до созвона
- Доля доведения до оплаты

## Контур
1. Контент-переход
2. Лид-магнит
3. Прогрев
4. Оффер
5. Follow-up
