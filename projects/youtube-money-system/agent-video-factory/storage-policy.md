# Правило хранения — Agent Video Factory

## Решение Дмитрия
Тяжёлые файлы не держим в памяти Hermes и не складываем в Obsidian.

## Где что хранится

### Яндекс.Диск — тяжёлые материалы
Путь:
`yandex:YouTube_Money_System/Agent_Video_Factory/`

Там храним:
- исходные видео и референсы;
- кадры;
- анимации;
- озвучки;
- готовые MP4;
- превью;
- архивы выпусков.

### Obsidian — только индекс и управление
Путь проекта:
`/root/obsidian-vault/projects/youtube-money-system/agent-video-factory/`

Там храним:
- роли агентов;
- pipeline (конвейер);
- планы;
- ссылки на файлы в Яндекс.Диске;
- статусы;
- решения;
- короткие отчёты.

### Hermes memory — только короткий указатель
В долговременной памяти храним только правило: тяжёлые ассеты YouTube-проекта лежат на Яндекс.Диске, а не в памяти/Obsidian.

## Структура на Яндекс.Диске

```text
YouTube_Money_System/Agent_Video_Factory/
├── 00_Index/
├── 01_Donors/
├── 02_Scripts/
├── 03_Voiceover/
├── 04_Frames/
├── 05_Animations/
├── 06_Renders/
├── 07_Thumbnails/
├── 08_Publish_Logs/
├── 09_Analytics/
└── 10_Archive/
```

## Операционное правило
1. Агент создаёт файл локально только временно.
2. После проверки файл переносится/копируется на Яндекс.Диск.
3. В Obsidian записывается ссылка/путь и короткое описание.
4. Локальные временные файлы удаляются после подтверждения, если они тяжёлые.
5. Удаление с Яндекс.Диска — только после явного подтверждения Дмитрия.
