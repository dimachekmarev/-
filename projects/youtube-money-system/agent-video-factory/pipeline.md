# Сквозной конвейер (pipeline) одного видео

## Общая схема
```
Donor Scout ──тема──→ Script Writer ──сценарий──→ [ЧЕЛОВЕК: вычитка] ──→ Visual Designer ──→ Video Editor
                             │                                              ↑                │
                             └──→ Voice Artist ────────────────────────────┘                │
                                                                                              ↓
                      Analytics Eye ←── Publisher ←── Packaging Designer ←── [ГОТОВОЕ ВИДЕО]
                             │
                             └──→ Controller ──→ [вечерний отчёт + правки]
```

## Этапы и ворота (GO/NO-GO)

### Этап 0: Входной контроль
- **Кто:** Controller
- **Что проверяет:** есть ли тема, доноры, Tone-of-Voice, целевой хронометраж.
- **GO:** всё заполнено → выдаёт задание Script Writer.
- **NO-GO:** не хватает данных → возврат Donor Scout.

### Этап 1: Сценарий
- **Кто:** Script Writer → человек (вычитка) → Controller
- **GO:** сценарий прочитан человеком вслух, правки внесены, хронометраж ±20%.
- **NO-GO:** синтетический язык, логические дыры → правки Script Writer.

### Этап 2: Озвучка
- **Кто:** Voice Artist → Controller
- **GO:** естественное звучание, ≤ 2 ошибок/мин, субтитры синхронизированы.
- **NO-GO:** роботизированный голос, ошибки → перенастройка TTS.

### Этап 3: Визуал
- **Кто:** Visual Designer → Controller
- **GO:** ≥ 90% тайм-кодов покрыто кадрами, ≥ 15% анимировано, стиль един.
- **NO-GO:** стиль прыгает, нет анимации (<15% кадров) → доработка Visual Designer.

### Этап 4: Монтаж
- **Кто:** Video Editor → Controller
- **GO:** видео проигрывается без ошибок, тайминг кадров = озвучка, субтитры синхронизированы.
- **NO-GO:** рассинхрон, битое видео → повторная сборка.

### Этап 5: Упаковка
- **Кто:** Packaging Designer → Controller
- **GO:** ≥ 3 заголовка, превью готово, SEO-пакет заполнен.
- **NO-GO:** заголовки слабые, превью не цепляет → итерация Packaging Designer.

### Этап 6: Публикация
- **Кто:** Publisher → Controller
- **GO:** видео доступно по ссылке, метаданные на месте, превью отображается.
- **NO-GO:** ошибка загрузки, блокировка YouTube → разбор ошибки, повтор.

### Этап 7: Аналитика (пост-публикация)
- **Кто:** Analytics Eye → Controller
- **GO:** отчёт содержит CTR, retention, просмотры, подписки + рекомендации на завтра.
- **NO-GO:** данные не собраны → проверка API, повторный сбор.

## Правила эскалации
- 2 NO-GO подряд на одном этапе → Controller поднимает флаг, видео откладывается до ручного разбора.
- Брак после публикации → Controller фиксирует инцидент в `control/incidents.md`, видео удаляется/исправляется.

## Целевые метрики конвейера
- Время от темы до публикации: < 90 минут (цель), < 4 часа (допустимо на старте).
- Видео в день: 3 Shorts + 1 Long/неделя (цель).
- Доля ручных правок: ≤ 20% текста сценария.
