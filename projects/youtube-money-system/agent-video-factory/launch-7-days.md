# 7-дневный план запуска Agent Video Factory

## День 1 — Инфраструктура и доступы
- [ ] Создать/верифицировать YouTube канал (если нет)
- [ ] Получить YouTube OAuth 2.0 (Google Cloud Project + Data API v3 + токен)
- [ ] Зарегистрировать ElevenLabs (или аналог), получить API ключ
- [ ] Выбрать генератор картинок: Midjourney / DALL-E / Higgsfield — зарегистрироваться, получить доступ
- [ ] Выбрать аниматор: RunwayML / Kaiber / ComfyUI — зарегистрироваться
- [ ] Установить vidIQ (или NextLev) в браузер
- [ ] Создать структуру папок agent-video-factory

## День 2 — Доноры и стратегия
- [ ] Donor Scout: собрать 20+ доноров по нишам Дмитрия (психология, привычки, деньги, отношения)
- [ ] Strategist: выбрать 2 фокус-ниши на неделю
- [ ] Скачать 3–5 сценариев конкурентов с тайм-кодами (vidIQ/NextLev)
- [ ] Подготовить промт-шаблон для Script Writer

## День 3 — Первый сценарий и озвучка
- [ ] Script Writer: написать 1 short-сценарий (60 сек)
- [ ] Человек: вычитка вслух, правки
- [ ] Voice Artist: озвучить финальный сценарий (ElevenLabs)
- [ ] Проверить субтитры

## День 4 — Визуал и монтаж
- [ ] Visual Designer: сгенерировать кадры по тайм-кодам (10–15 кадров)
- [ ] Анимировать ≥ 15% кадров (2–3 кадра)
- [ ] Video Editor: собрать черновое MP4

## День 5 — Упаковка и первая публикация
- [ ] Packaging Designer: 3 заголовка + 1 превью + SEO-пакет
- [ ] Controller: проверить все гейты → GO
- [ ] Publisher: загрузить видео на YouTube
- [ ] Проверить: видео доступно, метаданные на месте

## День 6 — Первый аналитический цикл
- [ ] Analytics Eye: подключить YouTube Analytics API
- [ ] Получить первый отчёт (CTR, retention, просмотры)
- [ ] Controller: вечерний отчёт + правки на завтра
- [ ] Donor Scout: собрать ещё 5–10 доноров

## День 7 — Ретроспектива и масштабирование
- [ ] Controller: полный отчёт за неделю (что сработало, что нет)
- [ ] Strategist: скорректировать ниши/форматы на неделю 2
- [ ] Script Writer: написать ещё 2 сценария
- [ ] Цель: выпустить 2–3 видео за день (если конвейер отлажен)
- [ ] План на неделю 2: выход на 2–3 видео/день
