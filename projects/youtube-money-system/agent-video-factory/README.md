# Agent Video Factory — паспорт проекта

## Цель
Построить конвейер из 10 агентов, который забирает донорский контент → генерирует сценарий → озвучку → визуал → монтаж → упаковку → публикует на YouTube → отдаёт аналитику. Цель: не менее 3 Shorts/день + 1 Long/неделю с минимальным ручным вмешательством.

## Референс
- Видео: [Claude + Ютуб = $30,000/Месяц](https://youtu.be/1oAzhMskoG0) — Артем Слабун
- Анализ системы: [[projects/youtube-money-system/plans/agent-production-system-analysis]]
- Основная доска: [[projects/youtube-money-system/plans/90-day-execution-board]]
