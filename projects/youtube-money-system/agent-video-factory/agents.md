# 10 агентов — карточки должностей

## Агент 1: Donor Scout (разведка доноров)
- **Роль:** ищет каналы-доноры в заданных нишах, собирает метаданные, вытягивает сценарии с тайм-кодами.
- **Вход:** ниша, ключевые слова, список конкурентов, vidIQ/NextLev.
- **Выход:** карточка донора (`donors/donor-YYYY-MM-DD.json`) + дайджест трендов на день (`donors/daily-trends.md`).
- **KPI:** ≥ 5 новых доноров/неделя; дайджест к 09:00 каждое утро.
- **Гейт приёмки:** у каждого донора заполнены: ссылка, ниша, формат, что берём, риск.
- **Первое задание:** собрать 20 доноров по нишам: психология, привычки, деньги, отношения.

## Агент 2: Strategist (стратег ниш)
- **Роль:** выбирает 2–3 фокус-ниши на неделю, решает пропорцию shorts/long, утверждает формат.
- **Вход:** данные Donor Scout, метрики канала, KPI-доска.
- **Выход:** `strategy/weekly-niche-focus.md` (ниши, форматы, цели).
- **KPI:** CTR > 4% по выбранным нишам; retention растёт неделя к неделе.
- **Гейт приёмки:** под каждую нишу ≥ 5 доноров, ниши не пересекаются.
- **Первое задание:** выбрать 2 фокус-ниши из пула Дмитрия на первую неделю.

## Агент 3: Script Writer (сценарист)
- **Роль:** пишет сценарий с покадровыми тайм-кодами, описанием визуала и пометкой «ручная вычитка».
- **Вход:** тема + сценарии доноров + Tone-of-Voice Дмитрия.
- **Выход:** `scripts/script-YYYY-MM-DD-topic.md` (текст, тайм-коды, визуальные подсказки).
- **KPI:** ≥ 2 сценария/день; ≤ 30% правок человеком после первой итерации.
- **Гейт приёмки:** человек прочитал вслух, неестественные фразы исправлены, хронометраж ±20% цели.
- **Первое задание:** написать 1 short-сценарий (60 сек) на тему «Как сформировать привычку за 21 день».

## Агент 4: Visual Designer (визуал/раскадровка)
- **Роль:** генерирует кадры по тайм-кодам, 15–20% анимирует, сохраняет с именами по секундам.
- **Вход:** сценарий с описаниями визуала + референсы стиля.
- **Выход:** `assets/video-YYYY-MM-DD/frames/` + `assets/video-YYYY-MM-DD/animations/`.
- **KPI:** покрытие ≥ 90% тайм-кодов кадрами; стиль един.
- **Гейт приёмки:** визуал соответствует тексту диктора в каждом кадре, стиль выдержан.
- **Первое задание:** сгенерировать 10 кадров для тестового сценария, 3 из них анимировать.

## Агент 5: Voice Artist (озвучка)
- **Роль:** превращает финальный сценарий в аудиодорожку + субтитры.
- **Вход:** сценарий после ручной вычитки + голосовой профиль.
- **Выход:** `assets/video-YYYY-MM-DD/voiceover.mp3` + `subtitles.srt`.
- **KPI:** озвучка готова ≤ 15 мин; ≤ 2 ошибок произношения на минуту.
- **Гейт приёмки:** голос естественный, паузы правильные, субтитры синхронизированы.
- **Первое задание:** озвучить тестовый сценарий через ElevenLabs (русский голос).

## Агент 6: Video Editor (монтажёр)
- **Роль:** собирает финальное MP4: дорожка озвучки + кадры по тайм-кодам + субтитры + (опционально) фоновая музыка.
- **Вход:** кадры, анимации, озвучка, субтитры.
- **Выход:** `output/video-YYYY-MM-DD-final.mp4`.
- **KPI:** рендер ≤ 2× хронометража; сборка без ручных правок.
- **Гейт приёмки:** видео проигрывается без ошибок, тайминг кадров = озвучка, субтитры синхронизированы.
- **Первое задание:** собрать тестовое видео из 10 кадров + озвучка (30–60 сек).

## Агент 7: Packaging Designer (упаковщик)
- **Роль:** создаёт 3–5 заголовков, превью (thumbnail), описание, теги, хештеги.
- **Вход:** готовое видео + метаданные (ниша, ключевые слова, CTA).
- **Выход:** `packaging/video-YYYY-MM-DD-seo.md` + `thumbnails/thumb-*.png`.
- **KPI:** CTR опубликованного > 4% через 48 часов.
- **Гейт приёмки:** заголовки A/B-готовы (если возможно), превью привлекает без кликбейта.
- **Первое задание:** сделать 3 заголовка + 1 превью для тестового видео.

## Агент 8: Publisher (публикатор)
- **Роль:** загружает видео на YouTube с метаданными по расписанию.
- **Вход:** видео + SEO-пакет + расписание.
- **Выход:** запись в `publish-log.md` (URL, время, статус).
- **KPI:** публикация ±15 мин от расписания; 0 ошибок загрузки.
- **Гейт приёмки:** видео доступно по ссылке, метаданные на месте, превью отображается.
- **Первое задание:** загрузить тестовое видео (после получения OAuth).

## Агент 9: Analytics Eye (аналитик)
- **Роль:** собирает метрики YouTube, vidIQ, формирует ежедневный отчёт и рекомендации.
- **Вход:** YouTube Analytics API / vidIQ данные.
- **Выход:** `analytics/daily-report-YYYY-MM-DD.md`.
- **KPI:** отчёт к 09:00 каждый день; рекомендации подтверждены цифрами.
- **Гейт приёмки:** отчёт содержит CTR, retention, просмотры, лайки, подписки, лиды + конкретные действия.
- **Первое задание:** подключить YouTube Analytics API, получить первый отчёт (даже пустой).

## Агент 10: Controller (контролёр качества)
- **Роль:** проходит по всем этапам конвейера, ставит GO/NO-GO, выдаёт сводный вечерний отчёт.
- **Вход:** выходы всех агентов за день.
- **Выход:** `control/daily-gate-log.md` (решения по каждому этапу + правки на завтра).
- **KPI:** 100% видео через гейт; ≤ 5% брака после публикации.
- **Гейт приёмки:** видео соответствует политике YouTube, нет шаблонного контента.
- **Первое задание:** создать шаблон `daily-gate-log.md`, прогнать первую пустую итерацию.
