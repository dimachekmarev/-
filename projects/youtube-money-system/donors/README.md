# Доноры контента — реестр

Формат одной записи (присылай мне так, я буду сразу сохранять):

```
ДОНОР: Название
ССЫЛКА: https://...
НИША: ...
ФОРМАТ: shorts / long / mixed
ЯЗЫК: ru / en / ...
ЗАЧЕМ: что именно берем (структуру, темп, подачу, хуки)
РИСК: low / medium / high
```

Важно:
- Копируем только модель (структура, формат, темп), НЕ копируем дословно контент.
- Для 18+ — отдельная пометка и compliance (правила платформ/законов).
