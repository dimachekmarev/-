## Схема монетизации

### Прямая реклама (основной доход)
- Цена за пост: 300–1000 ₽ (при 500–2000 подписчиков)
- Цена за пост: 2000–5000 ₽ (при 5000–15000 подписчиков)
- Источник заявок: Telega.in, прямое обращение в канал
- Средний доход с канала: 5 000–30 000 ₽/мес

### Партнерские программы
- Getcourse (онлайн-школы): 30–50% с продажи курса
- Skillbox: 15–25%
- Admitad: товары 3–30%
- Travelpayouts: тревел-услуги 4–7%

### Собственные продукты (3–6 месяц)
- PDF-гайд «10 AI-инструментов для бизнеса» — 990 ₽
- Мини-курс «AI-агент за 48 часов» — 4 900 ₽
- Консультация «Разбор твоего бизнеса под AI» — 9 900 ₽

## Рост подписчиков

### Неделя 1–2
- Взаимопиар с каналами схожей тематики (по 3–5 обменов в день)
- Комментинг в крупных каналах
- Размещение ссылки в описании всех каналов друг на друга (сетка)

### Неделя 3–4
- Реклама в других каналах (бюджет: 2000–5000 ₽ на тест)
- Розыгрыши с партнерами
- Бесплатные лид-магниты (PDF, чеклисты) для привлечения

### Далее
- Таргетированная реклама Telegram Ads
- SEO-оптимизация названий и описаний каналов
- Кросспостинг из VK/YouTube/Dzen → Telegram (приводит подписчиков)

## Контент-план

### 3 поста в день
1. Утро (09:00): полезный пост / обучение / кейс
2. День (14:00): новость / тренд / обсуждение
3. Вечер (20:00): вовлекающий контент / мем / вопрос

### Типы контента
- Текстовые посты с 1 картинкой
- Короткие видео (до 1 минуты)
- Файлы (PDF, чеклисты)
- Опросы
- Репосты из других каналов сетки

## Автоматизация

- Content Factory генерит тексты по расписанию
- Cross-poster рассылает контент по 10 каналам
- Бот публикует по расписанию через Telethon
- Analytics считает подписчиков и охваты через Telegram API
