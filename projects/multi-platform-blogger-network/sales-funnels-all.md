# Воронки продаж — 5 сценариев

## Воронка #1: Прямая продажа рекламы

### Как рекламодатель находит нас
1. Видит канал через Telega.in / Биржу ВК / Epicstars
2. Видит канал через поиск в Telegram/VK
3. Получает рекомендацию от другого рекламодателя
4. Видит наш пост с пометкой «размещаем рекламу»

### Сценарий сделки (24 часа)
```
Рекламодатель → Пишет в ЛС / на почту
→ Автоответ: «Привет! Форматы и цены: [ссылка на media-kit]»
→ Рекламодатель выбирает формат, присылает текст/ссылку
→ Согласование даты публикации
→ Предоплата 100% (если новый) или 50% (постоянный)
→ Публикация поста
→ Отчет: скриншот охватов через 24 часа
→ Рекламодатель доволен → повторное размещение
```

### Ценовая сетка (Telegram)
| Подписчики | Цена поста | Охват прогноз |
|---|---:|---:|
| 100–500 | 200 ₽ | 50–150 |
| 500–2000 | 500–800 ₽ | 150–600 |
| 2000–5000 | 1500–2500 ₽ | 600–1500 |
| 5000–15000 | 3000–7000 ₽ | 1500–5000 |
| 15000+ | 10000+ ₽ | 5000+ |

### Дополнительные форматы
- 2 поста (основной + напоминание через 24ч): +50%
- Нативная интеграция (авторский текст): +30%
- Видео-обзор продукта: x2 к цене поста
- Закреплённый пост на 48 часов: x1.5

## Воронка #2: Партнерский маркетинг (Affiliate)

### Сценарий
```
Контент → В посте зашита партнерская ссылка
→ Читатель переходит по ссылке
→ Покупает продукт/услугу в течение 30–90 дней (cookie)
→ Мы получаем комиссию
```

### Лучшие форматы для affiliate
1. Подборка «Топ-5 сервисов для X» — 3–5 партнерских ссылок
2. Обзор продукта «Я попробовал X, вот результаты»
3. Кейс «Как я заработал X с помощью Y»
4. Сравнение «A vs B: что лучше для X»
5. Промокод «Скидка 20% по моему коду»

### Приоритетные партнерки
| Партнерка | Продукт | Комиссия | Средний чек |
|---|---|---|---|
| Getcourse | Онлайн-школы | 30–50% | 15 000 ₽ |
| Skillbox | Курсы | 15–25% | 40 000 ₽ |
| Admitad | Товары, сервисы | 3–30% | — |
| Leadgid | Лиды банкам | 300–1500 ₽/лид | — |
| Travelpayouts | Авиабилеты, отели | 4–7% | 500$+ |

### План заработка (1 канал, 2000 подписчиков)
- 15 партнерских постов/мес
- Конверсия: 1% переходов → покупка
- 500 просмотров × 15 постов = 7500 просмотров
- 75 переходов, 1–2 покупки
- Доход: 2 000–10 000 ₽/мес с канала

```
→ После трипваера → Продажа курса (4 900 ₽)

```

## Воронка #4: B2B-клиенты на AI-автоматизацию

### Кто приходит
- Владельцы бизнеса, которые читают каналы про AI и автоматизацию
- Видят кейсы внедрения AI-агентов
- Хотят «такое же для своего бизнеса»

### Сценарий сделки
```
Клиент → Пишет в канал / на почту
→ Автоответ: «Опишите бизнес, я пришлю релевантный кейс»
→ Отправляем кейс из похожей ниши
→ Предлагаем аудит: 4 900 ₽ (разбор 1 бизнес-процесса)
→ После аудита → КП на внедрение: 35 000–120 000 ₽
→ Запуск проекта → Клиент доволен → Кейс в портфолио
```

### Каналы привлечения
- Telegram-каналы (посты с кейсами)
- VK (статьи про автоматизацию)
- YouTube (видео-разборы кейсов)
- Прямые касания (outreach) — после approval

## Воронка #5: Трафик-арбитраж (перелив)

### Схема
```
TikTok/Reels (вирусный контент)
→ В описании ссылка на Telegram-канал
→ Telegram-канал прогревает и продаёт
→ Часть трафика уходит на YouTube (длинный контент)
→ YouTube → описание ведёт на сайт/продукт
```

### Монетизация перелива
1. Бесплатный трафик из вирусных площадок → монетизация на своих площадках
2. Покупка рекламы на других каналах → перелив в свой → продажа рекламы дороже
3. SEO трафик из Дзен/YouTube → affiliate ссылки

### Пример цепочки
```
TikTok (100 000 просмотров)
→ 1% переходов = 1000 в Telegram
→ Telegram реклама: 1000 ₽/пост × 5 постов/мес = 5000 ₽/мес
→ ROI: потратили 0 на рекламу (органический TikTok), заработали 5000 ₽
```
