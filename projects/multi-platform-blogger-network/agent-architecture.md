# Архитектура AI-агентов для блогерской сетки

## Общая схема

```
Content Factory (генерация контента)
    ↓
Content Distributor (распределение по площадкам)
    ↓
┌──────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ Telegram │    VK    │ YouTube  │  Дзен    │  TikTok  │   Insta  │
│ Publisher │Publisher │Publisher │Publisher │Publisher │Publisher │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘
    ↓
Analytics Collector (сбор метрик)
    ↓
Revenue Tracker (доход по всем каналам)
    ↓
Owner Dashboard (ежедневный отчёт)
```

## Агенты

### 1. Content Generator Agent (Генератор контента)
- Берёт: темы, ниши, tone of voice
- Генерит: 30+ постов в день для всех площадок
- Форматы: текст, подписи к видео, сценарии Reels/Shorts
- Инструменты: Content Factory + AI-модели
- Расписание: ежедневно в 02:00 МСК

### 2. Cross-Platform Publisher Agent (Кросспостер)
- Берёт: готовый контент из Generator
- Адаптирует формат под каждую площадку
- Публикует через API / Telethon / браузерную автоматизацию
- Расписание: 3 раза в день (09:00, 14:00, 20:00)

### 3. Growth Agent (Рост подписчиков)
- Взаимопиар: ищет каналы для обмена постами
- Комментинг: оставляет полезные комментарии в крупных каналах
- Хештеги: подбирает трендовые хештеги для каждой площадки
- KPI: +100 подписчиков/день на всю сетку

### 4. Ad Sales Agent (Продажа рекламы)
- Отвечает на заявки рекламодателей (автоответ)
- Отправляет media-kit и цены
- Фиксирует бронирования в календаре
- Считает доход и напоминает о публикации
- KPI: обработать заявку за < 30 минут

### 5. Affiliate Manager Agent (Партнерки)
- Отслеживает лучшие офферы в партнерках
- Генерит партнерские посты с уникальными ссылками
- A/B тестирует форматы (обзор vs подборка vs кейс)
- Считает конверсию по каждой партнерке

### 6. Analytics Agent (Аналитика)
- Собирает метрики со всех площадок
- Считает охваты, подписчиков, доход
- Строит ежедневный дашборд
- Сигналит: какие каналы/темы растут, какие падают

### 7. Content Recycler Agent (Переупаковка контента)
- Берёт: старый популярный контент
- Переупаковывает: текст → видео, видео → карусель, пост → Reels
- Перепубликует: 1 успешный пост = 5 форматов на 5 площадках

## Технический стек

- Telegram API: Telethon (Python), 10+ аккаунтов
- VK API: vk-api (уже настроен)
- YouTube: Content Factory заливает видео
- Дзен: RSS + API (автопостинг)
- TikTok/Instagram: Browser Automation (BrowserAct) для публикации

## Cron-расписание

| Время | Задача | Агент |
|---|---|---|
| 02:00 | Генерация контента на день | Content Generator |
| 09:00 | Утренняя публикация | Cross-Platform Publisher |
| 10:00 | Сбор аналитики за вчера | Analytics Agent |
| 12:00 | Взаимопиар / рост | Growth Agent |
| 14:00 | Дневная публикация | Cross-Platform Publisher |
| 18:00 | Обработка заявок на рекламу | Ad Sales Agent |
| 20:00 | Вечерняя публикация | Cross-Platform Publisher |
| 22:00 | Ежедневный отчёт владельцу | Analytics → Owner Report |
