# Smart Agent Ai — Архитектура автоконтента по соцсетям

Цель: чтобы контент выходил автоматически, а ты только утверждал (или вообще только смотрел отчёт).

## 1) Принцип: один канал = один мини-агент

Почему так:
- у каждой платформы свой формат, тон и алгоритмы;
- один «универсальный» агент делает средний контент;
- отдельные агенты дают качество и стабильность.

## 2) Набор мини-агентов

### A. Idea Scout (поиск идей)
- Вход: тренды, вопросы аудитории, конкуренты, твои кейсы.
- Выход: 10 идей/неделя с приоритетом по деньгам.

### B. Brand Guardian (контроль стиля)
- Следит за единым тоном, обещаниями, позиционированием.
- Не даёт «уехать» в разный стиль.

### C. Telegram Writer Agent
- Пишет посты под Telegram: длиннее, глубже, с CTA в ЛС.

### D. Instagram Writer Agent
- Пишет короткие хуки, карусели, Reels-сценарии.

### E. YouTube/TikTok Script Agent
- Делает сценарии коротких видео + заголовки.

### F. LinkedIn/X Writer Agent
- Деловой тон (LinkedIn) и короткие мощные тейки (X).

### G. Repurposer Agent (репакер)
- Берёт 1 смысл (идею) и делает 6–10 форматов под разные сети.

### H. Publisher Agent
- Публикация по расписанию, очереди, дедлайны.

### I. Performance Analyst
- Раз в день/неделю отчёт: что зашло, что не зашло, что масштабировать.

## 3) Поток работы (без перегруза)

1. Idea Scout → генерит идеи
2. Brand Guardian → фильтрует
3. Writers по платформам → пишут контент
4. Repurposer → размножает форматы
5. Publisher → ставит в расписание
6. Analyst → возвращает фидбек для улучшения

## 4) Контент-матрица (минимум)

- Telegram: 1 пост/день
- Instagram: 1 Reels + 1 stories блок/день
- TikTok/Shorts: 1 видео/день
- LinkedIn/X: 1 пост/день

## 5) Что ты делаешь лично

Режим «минимум участия»:
- 15 минут в день: approve (утвердить) очередь на завтра.
- 30 минут в неделю: посмотреть отчёт, выбрать лучшие связки.

## 6) Как внедряем по шагам (без хаоса)

### Этап 1 (3 дня) — Telegram как главный канал
- Настроить мини-агентов: Idea Scout + Telegram Writer + Publisher.
- Добиться стабильных постов и лидов.

### Этап 2 (7 дней) — Подключить Instagram + Shorts
- Добавить Repurposer.
- Контент из Telegram превращать в Reels/Shorts.

### Этап 3 (7 дней) — LinkedIn/X + аналитика
- Добавить Analyst и weekly-ретро.

## 7) KPI (метрики) для денег

- Входящие ЛС в день
- Диагностики в неделю
- КП в неделю
- Оплаты в месяц

Контент-метрики вторичны. Главные — деньги и лиды.

## 8) Правила качества (лакшери-режим)

- Никакого постинга «ради постинга».
- Каждый пост должен вести к действию: АУДИТ / СТАРТ / СХЕМА.
- 1 оффер на неделю, не прыгать между темами.
- Еженедельно масштабировать только то, что дало лиды.

## 9) Следующий практический шаг

Собрать отдельный «паспорт автоматизации соцсетей»:
- какие сети подключаем первыми;
- точное расписание публикаций;
- какие mini-agents запускаются утром/днём/вечером;
- как выглядит ежедневный отчёт в Telegram.
