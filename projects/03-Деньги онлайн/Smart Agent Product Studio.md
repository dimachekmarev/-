# Smart Agent Product Studio

Статус: **рабочий документ / запущено после подтверждения Дмитрия 2026-06-14**  
Главная цель: быстро получить первые оплаты и приблизиться к минимуму **$5,000 к 2026-10-01** через пакетные AI-внедрения.

## 1) Позиционирование

**Smart Agent Product Studio** — продуктовая студия быстрых AI-внедрений для бизнеса.

Мы не продаём “консалтинг про AI”. Мы продаём понятный результат:
- больше заявок и диалогов;
- быстрее обработка входящих;
- меньше ручной рутины;
- готовые агенты, встроенные в ежедневный процесс клиента.

Формат продажи: пакетные офферы **Lite / Pro / VIP**.

## 2) Две стартовые ниши

### Ниша 1 — эксперты, консультанты, мини-школы

Кому продаём:
- коучи, наставники, консультанты;
- онлайн-школы и авторы платных продуктов;
- эксперты с Telegram / Instagram / YouTube;
- небольшие команды, где собственник сам пишет контент и отвечает в личке.

Главная боль:
- нет стабильного контента;
- заявки теряются в личных сообщениях;
- нет нормальной воронки и follow-up (повторных касаний);
- эксперт перегружен ручной упаковкой офферов, постами и ответами.

Что внедряем:
- контент-агент;
- агент упаковки офферов;
- DM/sales-ассистент (помощник по диалогам и продажам);
- follow-up цепочка;
- простой трекер лидов.

Измеримый результат:
- 7–14 дней контента;
- 1–3 оффера;
- 30–100 исходящих касаний;
- 5–20 квалифицированных диалогов;
- готовая система обработки заявок.

Почему это стартовая ниша:
- быстрый цикл сделки;
- легко показать результат за 3–7 дней;
- можно быстро получить первые кейсы.

### Ниша 2 — локальные высокочековые услуги

Кому продаём:
- стоматологии и косметологии;
- частные клиники;
- юридические услуги;
- ремонт, дизайн, строительство;
- риелторы и агентства недвижимости.

Главная боль:
- заявки приходят из разных каналов;
- менеджеры отвечают медленно;
- нет follow-up;
- часть лидов не дожимается;
- собственник не видит, где теряются деньги.

Что внедряем:
- агента первичной обработки заявок;
- FAQ-бота (ответы на частые вопросы);
- follow-up агента;
- скрипты ответов;
- трекер лидов и статусов;
- простую аналитику по воронке.

Измеримый результат:
- время ответа до 1–5 минут;
- 100% фиксация входящих заявок;
- 3–5 follow-up касаний по каждому лиду;
- понятная воронка: заявка → диалог → запись → оплата;
- рост конверсии в запись / консультацию.

Почему это стартовая ниша:
- высокая стоимость клиента;
- понятный ROI (возврат денег от внедрения);
- можно продавать не “AI”, а “не терять заявки”.

## 3) Пакетные офферы

### Lite — AI-аудит + быстрый агент

Цена: **$300–500**  
Срок: **2–3 дня**

Для кого:
- клиент хочет попробовать AI без большого бюджета;
- нет готовой системы;
- нужен быстрый видимый результат.

Что входит:
- короткая диагностика бизнеса и воронки;
- карта 3–5 процессов, где AI может дать быстрый эффект;
- настройка 1 простого агента;
- 1 набор промптов / скриптов;
- мини-инструкция для клиента;
- 30 минут созвона по внедрению.

Примеры агентов:
- генератор контент-плана;
- агент ответов на частые вопросы;
- агент квалификации лидов;
- агент follow-up сообщений;
- агент упаковки оффера.

Результат для клиента:
- 1 рабочий AI-агент;
- минимум 1 автоматизированный процесс;
- экономия 3–5 часов ручной работы в неделю;
- список следующих внедрений.

Цель для нас:
- быстрый первый платёж;
- вход в клиента;
- перевод в Pro.

### Pro — AI-воронка под ключ

Цена: **$1,200–2,000**  
Срок: **7–10 дней**

Для кого:
- есть лиды, но нет системы обработки;
- бизнес теряет заявки;
- собственник или команда перегружены ручной работой.

Что входит:
- аудит текущей воронки;
- проектирование AI-процесса;
- настройка 3–5 агентов;
- скрипты продаж / сообщений / follow-up;
- трекер лидов;
- базовая аналитика;
- обучение клиента;
- 7 дней поддержки после запуска.

Для экспертов:
- контент-агент;
- оффер-агент;
- DM/sales-агент;
- follow-up агент;
- агент анализа диалогов.

Для локальных услуг:
- агент первичного ответа;
- FAQ-агент;
- агент записи / квалификации;
- follow-up агент;
- агент отчёта по лидам.

Результат для клиента:
- 3–5 рабочих агентов;
- 1 собранная воронка;
- 30–100 обработанных лидов или касаний;
- 5–20 квалифицированных диалогов;
- понятный dashboard (экран метрик) по заявкам;
- снижение ручной нагрузки минимум на 5–10 часов в неделю.

Цель для нас:
- основной денежный продукт;
- 3 продажи Pro дают примерно **$3,600–6,000**.

### VIP — AI Sales/Ops System + сопровождение

Цена: **$3,000–5,000**  
Срок: **21–30 дней**

Для кого:
- бизнес хочет не разовое внедрение, а систему;
- есть команда, лиды и продажи;
- нужна автоматизация продаж, контента и операционки.

Что входит:
- стратегическая диагностика;
- карта процессов бизнеса;
- проектирование AI-операционной системы;
- настройка 7–12 агентов;
- интеграция с CRM / Google Sheets / Notion / Telegram;
- регламент работы команды;
- dashboard метрик;
- еженедельные улучшения;
- 4 недели сопровождения.

Примеры агентов:
- Lead Intake Agent (приём входящих лидов);
- Sales Follow-up Agent (дожим и повторные касания);
- Content Agent (контент);
- Offer Agent (офферы);
- CRM Update Agent (обновление CRM / таблицы);
- Client Support Agent (поддержка клиентов);
- Analytics Agent (аналитика);
- Manager Control Agent (контроль руководителя).

Результат для клиента:
- 7–12 рабочих агентов;
- единая система обработки лидов;
- ежедневный отчёт по продажам / заявкам;
- 100% фиксация входящих обращений;
- сокращение ручной работы на 10–20 часов в неделю;
- скорость ответа до 1–5 минут;
- план масштабирования на 90 дней.

Цель для нас:
- закрыть большую часть цели $5,000 одной сделкой;
- сделать сильный кейс;
- создать основу для ежемесячного сопровождения.

## 5) Воронка продаж

### Шаг 1 — лидогенерация

Каналы:
- личные сообщения в Telegram / LinkedIn / Instagram;
- посты в Telegram;
- комментарии под постами экспертов и владельцев бизнеса;
- холодные касания по нишевым спискам;
- рекомендации;
- мини-аудиты в подарок.

Дневная норма на старт:
- 30–50 новых лидов в базе;
- 10–20 первых сообщений;
- 5–10 follow-up;
- 2–5 диагностических диалогов.

### Шаг 2 — бесплатный мини-аудит

Оффер:
> Покажу 3 места, где AI-агенты могут сэкономить вам время или вернуть потерянные заявки.

Формат:
- 10–15 минут;
- голосовое / текст / короткий Loom;
- без сложной презентации.

Результат аудита:
- 3 проблемы;
- 3 быстрых AI-решения;
- рекомендация пакета Lite / Pro / VIP.

### Шаг 3 — диагностика

Цель:
- понять текущую воронку;
- найти деньги, которые теряются;
- показать конкретный результат;
- закрыть на пакет.

Вопросы:
- откуда приходят заявки?
- кто отвечает?
- как быстро отвечаете?
- где фиксируются лиды?
- сколько лидов теряется?
- какие задачи повторяются каждый день?
- сколько стоит один клиент?
- что будет ценнее: больше лидов, быстрее ответы или меньше рутины?

### Шаг 4 — продажа пакета

Логика:
- если клиент сомневается → Lite;
- если есть понятная воронка и заявки → Pro;
- если есть команда и деньги в процессе → VIP.

Ключевая формула:
> Мы не просто настроим AI. Мы соберём процесс, который каждый день помогает получать, обрабатывать и дожимать заявки.

### Шаг 5 — быстрый результат

После оплаты:
- день 1: диагностика и карта процесса;
- день 2–3: первый агент;
- день 4–7: воронка / интеграции / тесты;
- день 7–10: отчёт, кейс, допродажа.

### Шаг 6 — кейс и допродажа

Собрать после внедрения:
- что было до;
- что внедрили;
- сколько агентов;
- сколько времени экономит;
- сколько лидов обработано;
- какой следующий шаг.

Допродажа:
- Lite → Pro;
- Pro → VIP;
- VIP → monthly retainer (ежемесячное сопровождение).

## 6) Ежедневный регламент агентов

Формат дня: **4 часа, 10:00–14:00 MSK**.

### Strategy Agent

Задача:
- держит фокус на цели $5,000;
- выбирает приоритет дня;
- проверяет, не расползается ли проект.

Каждый день:
- утром выбирает 1 главный денежный фокус;
- вечером сверяет результат;
- отмечает риски и узкие места.

Выход:
- приоритет дня;
- решение: продавать / делать / дожимать / улучшать.

### Offer Architect

Задача:
- формирует офферы;
- улучшает Lite / Pro / VIP;
- адаптирует предложение под нишу.

Каждый день:
- обновляет 1–2 оффера;
- пишет короткие pitch-сообщения (короткие сообщения для продажи);
- готовит ответы на возражения.

Выход:
- готовый текст оффера;
- версия для экспертов;
- версия для локальных услуг.

### Traffic Agent

Задача:
- ищет лиды;
- собирает списки;
- готовит первые касания.

Каждый день:
- 30–50 новых лидов;
- сегментация по нише;
- подготовка персонализированных сообщений.

Выход:
- список лидов;
- статус: новый / написал / ответил / диагностика / отказ / сделка.

### Sales Closer Assistant

Задача:
- ведёт диалоги;
- квалифицирует лидов;
- помогает закрывать оплаты.

Каждый день:
- отвечает на входящие;
- делает follow-up;
- готовит предложения;
- фиксирует возражения.

Выход:
- количество диалогов;
- количество диагностик;
- количество предложений;
- потенциальная сумма сделок.

### Build Squad

Задача:
- собирает AI-агентов и автоматизации.

Каждый день:
- делает MVP для клиента;
- тестирует промпты;
- проверяет сценарии;
- готовит инструкции.

Выход:
- рабочий агент;
- ссылка / инструкция;
- список ошибок;
- версия 1.0 / 1.1 / 1.2.

### Ops Controller

Задача:
- следит за сроками, качеством и метриками.

Каждый день:
- проверяет pipeline (поток задач);
- считает лиды, диалоги, оплаты;
- фиксирует blockers (блокеры);
- готовит короткий отчёт.

Выход:
- daily report (короткий дневной отчёт);
- выручка;
- прогноз;
- что сделать завтра.

## 7) Ритм 4-часового фокус-блока

### Блок 1 — 30 минут: фокус дня

Определить:
- что продаём;
- кому пишем;
- какой результат нужен сегодня.

### Блок 2 — 90 минут: лидогенерация

Сделать:
- собрать лиды;
- отправить сообщения;
- сделать follow-up;
- назначить диагностики.

### Блок 3 — 90 минут: продажи / внедрения

Сделать:
- провести диагностики;
- закрыть оплату;
- собрать агента;
- показать клиенту быстрый результат.

### Блок 4 — 30 минут: отчёт

Зафиксировать:
- лиды;
- ответы;
- диагностики;
- оплаты;
- выручка;
- blockers;
- план на завтра.
