# Проект 03 — Деньги онлайн

Цель: максимальный онлайн-заработок через систему активного, полуавтоматического и пассивного/долгоиграющего дохода.

North Star goal: [[main-goal-2026-10-01]] — заработать и получить **минимум $5,000 к 2026-10-01**; это minimum target, а не потолок. Команда работает на максимум быстрее: первые деньги как можно раньше + построение устойчивых потоков дохода. Агентские идеи, новые KPI и запуски — только после approval Дмитрия.

## Фокус
- Не прыгать между проектами: все направления ведутся внутри **«03 — Деньги онлайн»**.
- Каждое направление оценивать по матрице: **скорость денег / автоматизация / риск / усилие**.
- Быстро доводить до денег, затем усиливать автоматизацию и долгосрочные активы.

## Типы денег

1. **Быстрый активный доход** — услуги, внедрения, консалтинг, диагностики, Product Studio / Smart Agent внедрения.
2. **Полуавтоматический доход** — шаблоны, PDF, мини-продукты, автоворонки, простые продуктовые линейки.
3. **Пассивный / долгоиграющий доход** — YouTube, SEO, микро-SaaS, контентные активы, подписочные или evergreen-механики.

## Рабочие метрики (proposed / требуют approval Дмитрия)
- Лиды в день
- Квалифицированные диалоги
- Оплаты
- Выручка
