# Smart Agent Ai — Боевая настройка соцсетей (Production)

Статус: ACTIVE
Режим: лакшери / автопилот / контроль по деньгам

## 1) Каналы и роли mini-agents

### Telegram Agent
- Формат: экспертные посты + кейсы + оффер
- Частота: 1 пост/день
- Цель: входящие в ЛС (код: АУДИТ)

### Instagram Agent
- Формат: Reels-сценарий + карусель + stories
- Частота: 1 единица/день
- Цель: переход в ЛС/директ

### TikTok/Shorts Agent
- Формат: короткие сценарии 20–40 сек
- Частота: 1/день
- Цель: охват + лид-магнит

### LinkedIn/X Agent
- Формат: короткие тезисы + кейс/инсайт
- Частота: 1/день
- Цель: B2B диалоги и партнёрства

### Idea Scout Agent
- Частота: 1 раз в сутки
- Выход: 5 идей контента + 2 идеи оффера

### Repurposer Agent
- Делает из 1 основной идеи версии под все платформы

### Publisher Agent
- Собирает и выдаёт «очередь публикаций на завтра»

### Performance Analyst Agent
- Ежедневно: короткий отчёт
- Еженедельно: что масштабируем, что удаляем

---

## 2) Ежедневный регламент (МСК)

- 09:00 — Idea Scout: идеи на день
- 10:00 — Writers: создание контента
- 11:30 — Repurposer: адаптация под платформы
- 12:00 — Publisher: формирование очереди
- 21:00 — Performance Analyst: отчёт по лидам/деньгам

Роль Дмитрия:
- 15 минут в 12:00: approve очереди
- 0–10 минут в 21:00: посмотреть отчёт

---

## 3) KPI (деньги-центрично)

Основные:
- Входящие ЛС (целевые)
- Диагностики
- Отправленные КП
- Оплаты

Вторичные:
- Охват
- ER
- Подписки

Правило:
Контент без лидов/диалогов 7 дней подряд — отключается.

---

## 4) Контент-рамка 40/40/20

- 40% кейсы (цифры до/после)
- 40% боль+решение (прикладные разборы)
- 20% личный бренд/закулисье

Каждый пост обязан иметь 1 CTA:
- АУДИТ / СТАРТ / СХЕМА

---

## 5) Недельный цикл

Пн: оффер недели
Вт: кейс
Ср: разбор ошибки
Чт: демонстрация процесса
Пт: отзыв/пруф
Сб: дедлайн оффера
Вс: weekly review + план следующей недели

---

## 6) Анти-хаос правила

1. Один оффер в неделю.
2. Не публиковать «воду» без цели.
3. Не менять сразу всё — только 1 переменная за тест.
4. Масштабировать только победителей по KPI.

---

## 7) Паспорт контроля

- Daily report: 21:00 МСК
- Weekly strategy report: воскресенье 20:30 МСК
- Месячный отчёт: 1-е число, 11:00 МСК

Формат daily:
- Лиды: X
- Диагностики: Y
- КП: Z
- Оплаты: N
- Что улучшить завтра: 3 пункта
