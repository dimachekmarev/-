# Site Publish Queue — Smart Agent AI

Эта папка хранит готовые SEO-пакеты для сайта, если автоматическая публикация в WordPress недоступна.

## Почему может появиться файл в очереди
- Не заданы доступы WordPress API
- Временная ошибка API
- Ошибка авторизации

## Что нужно для автопубликации в WordPress
Переменные окружения на сервере Hermes:
- `WP_SITE_URL`
- `WP_USER`
- `WP_APP_PASSWORD`

После добавления доступов агент автоматически начнёт отправлять посты как draft и давать URL/ID в ежедневном отчёте.
