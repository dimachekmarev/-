# PDF-отчёт: ускорение сайта, SEO и доверие поиска

PDF: [[Smart-Agent-AI-site-speed-seo-checklist.pdf]]

Ключевые проблемы:
- no-store/no-cache на публичных страницах;
- PHPSESSID для анонимного посетителя;
- много CSS/JS;
- mixed content http:// внутри https://;
- тяжёлые изображения;
- sitemap в robots.txt через http;
- нужно усилить E-E-A-T: автор, компания, контакты, кейсы, перелинковка.

Короткий план:
1. Кеш + PHPSESSID + http→https + robots/sitemap.
2. WebP/AVIF + lazy loading + правильные размеры карточек.
3. Чистка CSS/JS + defer.
4. Schema/canonical/Open Graph.
5. Страницы доверия + перелинковка + кейсы.
