# Референс по веб-дизайну (вдохновение)

## Основной источник
- https://godly.website/

## Правило для агентной разработки сайтов
- Для всех задач по дизайну/сборке сайтов Smart Agent AI использовать визуальное вдохновение от Godly.
- Не копировать 1-в-1: адаптировать под бренд Smart Agent AI, цели бизнеса и SEO.

## Что именно брать как ориентир
- Премиальная визуальная подача (плотная типографика, сильный hero-блок).
- Акцент на современный UI, контраст, аккуратные градиенты, карточки и микро-детали.
- Чистая структура лендинга: оффер → кейсы/решения → процесс → CTA.

## Ограничения
- Контент и структура должны быть оригинальными для Smart Agent AI.
- Сохранять скорость загрузки и мобильную адаптацию.
- Сразу закладывать SEO-основу (title/description, OG, schema, FAQ).
