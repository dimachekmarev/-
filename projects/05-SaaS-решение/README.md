# Проект 05 — SaaS-решение

Цель: создать и масштабировать SaaS-модель (подписка) как отдельный денежный актив.

## Фокус
- Выбор ниши и боли клиента
- Быстрый MVP (минимально рабочий продукт)
- Подписочная модель (monthly recurring revenue = регулярная выручка в месяц)
- Удержание клиентов (retention = удержание)

## KPI
- Активные подписки
- MRR (ежемесячная повторяющаяся выручка)
- CAC (стоимость привлечения клиента)
- Churn (отток клиентов)
- Retention (удержание)

## Этапы
1. Идея и валидация спроса
2. MVP за 7–14 дней
3. Первые 10 платящих пользователей
4. Улучшение продукта по обратной связи
5. Масштаб через контент и партнёрки
