# 06 — UFORMA медодежда ⚔️

## Ситуация
Семейный бизнес. После развода бывшая жена забрала 90%:
- **Производство** — у неё
- **У Дмитрия:** 1 магазин + VK
- Сайта нет, маркетплейсов нет

Задача: стартовать с VK, построить сайт круче чем у неё, порвать по продажам и контенту.

## Анализ конкурента (u-forma.ru)

### Что у неё
- **Платформа:** Bitrix (тяжёлая, медленная)
- **Каталог:** женская (12 категорий), мужская (5), поварская (2)
- **116+ товаров**
- **Разделы:** Premium, Sale, Size+, «Одеть клинику», подарочные сертификаты
- **Индивидуальный пошив**, корпоративные заказы
- **Шоурумы:** Москва (Серпуховская), СПб (2 адреса), Краснодар (2 адреса)
- **Контакты:** 8-800, info@u-forma.ru, VK, Telegram, YouTube, WhatsApp
- **ИП Дымченко Екатерина Юрьевна**

### Слабые места конкурента
1. Bitrix — медленно, неадаптивно, старая вёрстка
2. Нет современных визуальных решений (hero, видео, анимации)
3. SEO-описание слабое («недорого»)
4. Нет нормальной мобильной вёрстки
5. VK — формальное присутствие, не системное
6. Нет автоворонок, нет чат-ботов
7. Маркетплейсы не используются

### Наши преимущества
1. **Современный сайт:** Next.js — быстрый, продающий, адаптивный
2. **Системный VK:** уже есть готовая система автопостинга (скрипт, визуалы, Drive)
3. **Агенты:** контент, каталог, продажи, аналитика — автоматом
4. **SEO:** современные статьи, быстрая индексация
5. **Маркетплейсы:** можем выйти туда, где её нет

## Боевой план (фазы)

### Фаза 1 — БЫСТРЫЙ СТАРТ (VK + лендинг)
**Цель:** начать получать заявки через 7 дней

1. **VK-магазин:**
   - Оформить сообщество: товары, витрина, контакты
   - Подключить систему автопостинга (2 поста/день)
   - Запустить таргет на врачей и клиники

2. **Одностраничный лендинг:**
   - Геройский блок с УТП
   - Каталог из 12–15 топовых позиций
   - Быстрый заказ через Telegram/WhatsApp
   - Блоки доверия, отзывы, доставка

3. **Лидогенерация:**
   - Собрать базу клиник и салонов (50+ контактов)
   - Прямые касания

### Фаза 2 — ПОЛНОЦЕННЫЙ САЙТ
**Цель:** запустить сайт круче u-forma.ru

1. Next.js, продающий дизайн
2. Полный каталог с фильтрами
3. Корзина + оплата
4. SEO-оптимизация
5. Блог: статьи для врачей и клиник

### Фаза 3 — МАРКЕТПЛЕЙСЫ
**Цель:** Wildberries / Ozon / Яндекс Маркет

## Что нужно от Дмитрия сейчас
1. Ссылка на VK-сообщество UFORMA
2. Фото товаров (хоть телефоном — обработаем)
3. Ассортимент: что есть в магазине прямо сейчас
4. Цены
5. Домен (если есть) или придумаем новый
