# 06 — UFORMA медодежда

## Коротко
**UFORMA** — проект по продаже медицинской одежды.

Цель проекта: упаковать направление в понятную онлайн-систему продаж: сайт / карточки товаров / контент / лиды / заказы / аналитика.

## Статус
- Статус: **intake** (первичное добавление проекта)
- Дата добавления: 2026-06-14
- Владелец: Дмитрий Чекмарев
- Нужно уточнить: ассортимент, цены, география, каналы продаж, текущие активы.

## Что нужно собрать от владельца
1. Сайт или домен, если уже есть.
2. Логотип / фирменные цвета / фото продукции.
3. Ассортимент:
   - халаты;
   - костюмы;
   - брюки;
   - топы;
   - обувь;
   - аксессуары.
4. Цены и маржинальность.
5. География доставки.
6. Контакты для заказов: Telegram / WhatsApp / телефон / сайт.
7. Где уже есть аккаунты: VK, Instagram, Telegram, маркетплейсы.

## Первичный план запуска

### 1) Упаковка продукта
- Разделить ассортимент по категориям.
- Сделать 3–5 ключевых офферов:
  - базовый комплект;
  - премиум комплект;
  - корпоративный заказ для клиник / салонов.
- Подготовить тексты: выгода, ткань, размерная сетка, уход, доставка.

### 2) Витрина продаж
- Подготовить лендинг / каталог.
- Сделать карточки товаров.
- Добавить быстрый заказ через мессенджеры.
- Добавить блоки доверия: качество ткани, размеры, обмен, доставка.

### 3) Контент и трафик
- VK / Telegram / Instagram-контент.
- SEO-статьи:
  - как выбрать медицинский костюм;
  - какие ткани лучше для медодежды;
  - что купить клинике для персонала;
  - размерная сетка и уход.
- Лидогенерация по клиникам, косметологиям, стоматологиям, салонам.

### 4) Автоматизация агентами
- Агент контента: посты и статьи.
- Агент каталога: карточки товаров.
- Агент продаж: лид-листы и первые касания.
- Агент аналитики: метрики, заявки, конверсия.

## KPI на старт
- 20 товаров заведены в каталог.
- 10 постов подготовлены.
- 3 оффера собраны.
- 50 лидов клиник / салонов собраны.
- Первые 10 касаний отправлены.

## Следующий шаг
Получить от Дмитрия вводные по бренду UFORMA: сайт, фото, ассортимент, цены, контакты и текущие площадки.
