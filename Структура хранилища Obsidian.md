# Структура хранилища Obsidian

Путь к хранилищу: `/root/obsidian-vault`

```text
obsidian-vault/
├── Home.md                              # Главный индекс
├── company/                             # Информация о компании
│   ├── Smart Agent AI.md                # Карточка компании
│   └── Методология.md                   # Подход к автоматизации
├── clients/                             # Клиенты и CRM
│   ├── Клиенты.md                       # Каталог клиентов
│   └── CRM.md                           # Воронка и статусы
├── projects/                            # Проекты
│   └── Проекты.md                       # Активные и завершённые
├── agents/                              # Каталог агентов
│   └── Агенты.md                        # Активные агенты и роли
├── automation/                          # Исследования, архитектуры, готовые решения
│   └── GitHub Research - Content Factory.md
├── meetings/                            # Встречи и daily-заметки
├── learning/                            # AI-образование, материалы
└── templates/                           # Шаблоны
    └── Daily Note.md                    # Ежедневная заметка
```

## Правило использования
- Любая работа по клиенту → обновить `clients/`
- Любой новый проект → добавить/обновить в `projects/`
- Любая встреча → заметка в `meetings/`
- Любой новый агент → запись в `agents/Агенты.md`
- Ежедневно → заполнять daily note
