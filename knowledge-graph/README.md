# Knowledge Graph — Внутренняя карта знаний Hermes

Построен на основе [Graphiti](https://github.com/getzep/graphiti) — построение графа без нейросети.

## Что это
132+ нод заметок Obsidian, связанных через `[[wiki-ссылки]]`. Используется для поиска связей между проектами, клиентами, агентами без сжигания токенов на LLM.

## Как обновить
```bash
cd /root/obsidian-vault/knowledge-graph
python3 build-kg.py
```

## Структура
```
knowledge-graph/
├── build-kg.py          # Скрипт сборки графа
├── knowledge-graph.json  # Граф (ноды + связи)
└── README.md            # Этот файл
```

## Как Hermes использует
При поиске связей («какой проект связан с клиентом X?») Hermes загружает `knowledge-graph.json` и идёт по связям без LLM — только поиск по JSON.

## Экономия
- Без графа: ~500 токенов за поиск связей через LLM
- С графом: ~10 токенов за парсинг JSON
- Экономия: **40–50x**
