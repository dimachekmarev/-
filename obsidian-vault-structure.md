# Obsidian Vault Structure

Путь к хранилищу: `/root/obsidian-vault`

```text
obsidian-vault/
├── Home.md
├── company/
│   ├── Smart Agent AI.md
│   └── Методология.md
├── clients/
│   ├── Клиенты.md
│   └── CRM.md
├── projects/
│   ├── Проекты.md
│   └── youtube-money-system/
├── agents/
│   └── Агенты.md
├── automation/
├── meetings/
├── learning/
├── templates/
│   ├── Daily Note.md
│   └── Insight.md
└── db/
    └── smart-agent.db
```

## Правило использования
- Любая работа по клиенту → обновить `clients/`
- Любой новый проект → добавить/обновить в `projects/`
- Любая встреча → заметка в `meetings/`
- Любой новый агент → запись в `agents/Агенты.md`
- Ежедневно → daily note
