# Как подключен мой Obsidian (Smart Agent AI)

## 1) Где физически хранятся заметки
- Локально на сервере: `/root/obsidian-vault`
- Это и есть рабочая папка Obsidian (vault / хранилище).

## 2) Куда идет резерв и синхронизация
- GitHub репозиторий: `git@github.com:dimachekmarev/-.git`
- Ветка: `main`
- То есть фактически заметки хранятся:
  - в локальной папке на сервере,
  - и копией в GitHub.

## 3) Как это синкается автоматически
- На сервере настроен авто-push (автоматическая отправка изменений) каждые 30 минут.
- На Mac в Obsidian Git настроен auto-pull (автоматическое получение изменений) каждые 30 минут.

Итог: ты работаешь с одним хранилищем, а GitHub держит актуальную копию.

## 4) Простая схема
1. Пишешь заметку в Obsidian
2. Изменения попадают в папку `/root/obsidian-vault`
3. Автоматически отправляются в GitHub
4. На другом устройстве подтягиваются обратно

## 5) Что это дает
- Не теряются заметки (бэкап / резервная копия)
- Можно работать с разных устройств
- Есть история изменений через GitHub
